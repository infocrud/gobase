# GoBase — Financial Model

## Assumptions

| Assumption | Value |
|---|---|
| Average revenue per paying user (ARPU) | $45/mo (blended Starter/Pro mix) |
| Free-to-paid conversion rate | 3% |
| Monthly churn (paying customers) | 4% |
| Gross margin | 82% (infra ~18% of revenue) |
| CAC (developer-led, community) | $80 |
| LTV at 4% churn | ~$1,125 |
| LTV:CAC ratio | **~14:1** |
